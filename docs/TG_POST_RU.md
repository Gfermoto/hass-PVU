# Turkov + Home Assistant: релиз автоматики, которой можно доверять

Собрал и выкатил рабочий blueprint для ПВУ Turkov:
<https://github.com/Gfermoto/hass-PVU>

Это не просто "переключалка режимов", а автоматика для реальной жизни:

- держит температуру по day/night/away сценариям;
- управляет скоростью вентилятора по качеству воздуха (`low/medium/high`);
- не отправляет неподдерживаемые режимы устройству;
- защищает зимой: не уводит в опасный `fan_only` в холод;
- устойчиво работает с разными сущностями присутствия;
- в develop-версии есть диагностика (`debug_mode`) и безопасная политика при неизвестной наружной температуре.

Отдельно сделал 2 варианта:

- `pvu_min.yaml` — стабильная база;
- `pvu.yaml` — развиваемая версия с новыми фичами.

Если у вас Turkov + HA — можно брать и запускать, дальше тонко подстроить пороги под свой дом/квартиру.
